syncstate

syncstate

Reports how far the node has synced: the block height DigiByte Core has reached together with the DigiAsset chain analyzer's processing state. Use it to tell whether the node is fully synced and safe to query.

Result

NameTypeDescription
countnumericBlock height DigiByte Core is currently synced to.
syncnumericDigiAsset processing state. Negative numbers indicate how many blocks behind DigiAsset processing is (anything more than 120 behind is unsafe to use). 0 = fully synced; 1 = stopped; 2 = initializing; 3 = rewinding; 4 = optimizing (only occurs on first wallet sync, processed in sections so the node can enter and leave this state several times).

Examples

DigiAssetWindows-cli.exe syncstate

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "syncstate", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
