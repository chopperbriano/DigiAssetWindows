getoldstreamkey

getoldstreamkey key

Legacy compatibility shim that emulates the old DigiAsset Stream service: given a stream key it queues the request via OldStream. Deprecated and retained only for old clients; it should not be used by new projects.

Arguments

#NameTypeRequiredDescription
1keystringrequiredThe stream key to request. An integer is also accepted and converted to its string form.

Result

NameTypeDescription
resultnumericThe number of items in the job queue.

Examples

DigiAssetWindows-cli.exe getoldstreamkey mystreamkey

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getoldstreamkey", "params": ["mystreamkey"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
