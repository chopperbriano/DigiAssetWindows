shutdown

shutdown

Brings the node to a safe stopping point by halting the chain analyzer and the IPFS subsystem, so no processing is left mid-write, then logs a CRITICAL "Safe to shut down" message. Note: this does not exit the process itself; the operator or host still terminates it.

Result

NameTypeDescription
resultbooleanAlways true once the background workers have been stopped. The response is not cached.

Examples

DigiAssetWindows-cli.exe shutdown

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "shutdown", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
