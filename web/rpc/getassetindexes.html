getassetindexes

getassetindexes assetId

Returns the list of node-specific assetIndex values that belong to a given assetId. Most assetIds map to a single index, but a reissued id can map to several.

Arguments

#NameTypeRequiredDescription
1assetIdstringrequiredThe assetId to look up.

Result

An array of assetIndex values.

NameTypeDescription
assetIndexnumericA node-specific asset index belonging to the given assetId.

Examples

DigiAssetWindows-cli.exe getassetindexes La8VkHTqAg9rn3XPFdXX8jK9BeQaSl9Rcxvhu5

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getassetindexes", "params": ["La8VkHTqAg9rn3XPFdXX8jK9BeQaSl9Rcxvhu5"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
