resyncmetadata

Re-pins all IPFS metadata for every Permanent Storage Pool the node is subscribed to. Returns immediately; the actual downloads happen asynchronously in the background and will likely take a while to finish.

Result

NameTypeDescription
(result)boolAlways true. This confirms the re-pin was triggered; it does not mean the files have finished downloading.

Examples

DigiAssetWindows-cli.exe resyncmetadata

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "resyncmetadata", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
