getipfscount

getipfscount

Returns the number of IPFS jobs currently queued in the local database, letting operators gauge the node's pending pin/fetch backlog. The result is never cached.

Result

NameTypeDescription
resultnumericThe number of IPFS jobs in the queue.

Examples
