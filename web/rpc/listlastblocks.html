listlastblocks

listlastblocks limit (start)

Returns the most recently processed blocks from the node's database. Useful for showing chain-sync progress in clients and dashboards.

Arguments

#NameTypeRequiredDescription
1limitnumericrequiredNumber of block records to return.
2startnumericoptionalBlock height to start from, counting backwards. Defaults to the newest block.

Result

An array of block objects, each with the following fields:

NameTypeDescription
heightnumericBlock height.
hashstringBlock hash.
timenumericBlock timestamp.
algonumericMining algorithm used for the block.

Examples

DigiAssetWindows-cli.exe listlastblocks 10

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "listlastblocks", "params": [10]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
