listassetissuances

listassetissuances assetId

Returns every issuance transaction recorded for a given asset, covering the initial issuance and any later re-issuances.

Arguments

#NameTypeRequiredDescription
1assetIdstringrequiredThe asset identifier to look up issuances for.

Result

An array of objects, one per issuance transaction, each with the following fields:

NameTypeDescription
assetIndexnumericInternal database index for this specific asset issuance.
txidstringTransaction id of the issuance.
amountnumericQuantity of the asset issued in this transaction.
heightnumericBlock height at which the issuance was confirmed.
cidstringIPFS content id of the asset metadata for this issuance.

Examples

DigiAssetWindows-cli.exe listassetissuances La3iiK783Td9ELwc3RTNBFvGjntUpLTeh3F5uH

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "listassetissuances", "params": ["La3iiK783Td9ELwc3RTNBFvGjntUpLTeh3F5uH"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
