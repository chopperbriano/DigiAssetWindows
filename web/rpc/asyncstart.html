asyncstart

asyncstart method (param1 param2 ...)

Queues an RPC call to run in the background instead of blocking the caller, so long-running requests can be started and collected later. The named method is executed by a background worker and its JSON result is cached on disk; retrieve it with asyncget and discard it with asyncclear (using the same parameters).

Arguments

#NameTypeRequiredDescription
1methodstringrequiredName of the RPC method to run in the background.
2+param1, param2, ...anyoptionalAny parameters to pass to the named method, in order. These are forwarded to the target method exactly as given.

Result

NameTypeDescription
(result)numericThe number of jobs currently in the async job queue (including the one just added).

Examples

DigiAssetWindows-cli.exe asyncstart algostats 0

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "asyncstart", "params": ["algostats", 0]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
