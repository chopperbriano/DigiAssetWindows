getexchangerates

getexchangerates (height)

Returns the list of on-chain DigiAsset exchange rates published by oracle addresses. With no argument it returns the current rates; with a block height it returns the rates as they were at that height. Throws if the height is above the current sync height, or if no height is given and sync is more than 120 blocks behind.

Arguments

#NameTypeRequiredDescription
1heightnumericoptionalBlock height to report the exchange rates as of. If omitted, the current sync height is used.

Result

Returns an array of objects, each with the following fields. DGB has an implied value of 100000000 and is never included. To convert use inputAmount*inputRate/outputRate.

NameTypeDescription
heightnumericThe block height this exchange rate was recorded at.
addressstringThe oracle address this exchange rate was recorded at.
indexnumericWhich index the recorded value was at (0 to 9).
valuenumericThe exchange rate value.

Examples

DigiAssetWindows-cli.exe getexchangerates

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getexchangerates", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
