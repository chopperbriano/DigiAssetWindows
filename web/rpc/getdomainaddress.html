getdomainaddress

getdomainaddress domain

Resolves a DigiByte-Domain name to the DigiByte address that currently controls it. Returns an error if the domain is unknown or has been revoked.

Arguments

#NameTypeRequiredDescription
1domainstringrequiredThe DigiByte-Domain name to resolve.

Result

NameTypeDescription
addressstringThe DigiByte address currently associated with the domain.

Examples

DigiAssetWindows-cli.exe getdomainaddress example.dgb

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getdomainaddress", "params": ["example.dgb"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
