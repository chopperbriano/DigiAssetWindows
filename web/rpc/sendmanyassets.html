sendmanyassets

sendmanyassets sends ( options )

Sends DigiAssets to several recipients in a single transaction. The wallet picks which asset UTXOs to spend, sends all asset change back to the wallet, and pays the transaction fee in DigiByte. Up to 31 send entries are allowed (the protocol requires asset outputs to sit in the first 32 outputs).

Arguments

#NameTypeRequiredDescription
1sendsarrayrequiredArray of 1 to 31 objects, one per recipient. Each object has "address" (string, recipient address or DigiByte domain), "asset" (assetId string or assetIndex integer; assets with more than one index must use assetIndex) and "amount" (integer, decimal number or numeric string, in display units).
2optionsobjectoptionalObject with optional member "changeAddress" (string) the address asset change should go to, defaulting to a new wallet change address.

Result

The txid of the broadcast transaction.

NameTypeDescription
txidstringTransaction id of the broadcast transaction.
