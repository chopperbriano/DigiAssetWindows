getaddressholdings

getaddressholdings address

Returns all DigiAsset balances currently held by a single address as a map of assetIndex to quantity. DigiByte (assetIndex 1) is included only if the node was run with storenonassetutxo=1.

Arguments

#NameTypeRequiredDescription
1addressstringrequiredThe DigiByte address whose holdings to look up.

Result

An object whose keys are assetIndex values and whose values are the quantity held of that asset.

NameTypeDescription
<assetIndex>numericQuantity of the asset (identified by its assetIndex key) held by the address. AssetIndex 1 is DigiByte.

Examples

DigiAssetWindows-cli.exe getaddressholdings dgb1q...address

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getaddressholdings", "params": ["dgb1q...address"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
