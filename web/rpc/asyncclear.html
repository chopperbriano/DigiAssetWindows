asyncclear

asyncclear method (param1 param2 ...)

Deletes the cached result of an async RPC job so the call can be re-run. Call it with the same method name and parameters originally passed to asyncstart / asyncget.

Arguments

#NameTypeRequiredDescription
1methodstringrequiredName of the method whose cached result should be cleared.
2+param1, param2, ...anyoptionalThe same parameters that were passed to asyncstart, in the same order (used to identify the cached job).

Result

NameTypeDescription
(result)booltrue if a cached result existed and was deleted; false if there was no cached result to clear.

Examples

DigiAssetWindows-cli.exe asyncclear algostats 0

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "asyncclear", "params": ["algostats", 0]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
