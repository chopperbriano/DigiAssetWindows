getaddresskyc

getaddresskyc address

Returns on-chain KYC (know-your-customer) verification data for a single address. Verified addresses include a country code and either a name or a privacy-preserving hash; unverified addresses return only the address.

Arguments

#NameTypeRequiredDescription
1addressstringrequiredThe DigiByte address to look up KYC data for.

Result

An object. The country, name and hash fields are present only if the address has been KYC verified. Exactly one of name or hash is returned, never both.

NameTypeDescription
addressstringThe requested address, echoed back.
countrystringOnly if verified. ISO 3166-1 alpha-3 country code.
namestringOnly if verified and not anonymous. The verified person's name. Omitted when the address is verified but left anonymous.
hashstringOnly if verified and anonymous. SHA-256 of the person's full name plus a pin they provided at verification time. Returned instead of name.

Examples

DigiAssetWindows-cli.exe getaddresskyc dgb1q...address

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getaddresskyc", "params": ["dgb1q...address"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
