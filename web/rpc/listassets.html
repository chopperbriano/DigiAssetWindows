listassets

listassets ( count startIndex basic filter )

Lists DigiAssets issued on chain in ascending issuance order, starting from a given asset index. Optionally filters by Permanent Storage Pool (PSP) membership and returns either a basic summary or the full asset record.

Arguments

#NameTypeRequiredDescription
1countnumericoptionalMaximum number of records to return. Defaults to unlimited.
2startIndexnumericoptionalAsset index to start listing from. Defaults to 1.
3basicbooleanoptionalIf true (default) returns a basic summary per asset; if false returns the full DigiAsset JSON.
4filterobjectoptionalFilter object. Supports a psp member: a boolean (true = only assets in a PSP, false = only assets not in a PSP) or an integer (only assets in that specific PSP). Filtering may make the returned count differ from the requested count and can yield empty pages.

Result

An array. In basic mode each element is an object with the fields below; when basic is false each element is the full DigiAsset JSON (see getassetdata).

NameTypeDescription
assetIndexnumericNode-specific index of the asset (only valid on this node).
assetIdstringThe asset ID.
cidstringThe CID associated with the asset.
heightnumericThe block height at which the asset was issued.

Examples

DigiAssetWindows-cli.exe listassets 10 1

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "listassets", "params": [10, 1]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
