version

Returns the running build's version string so clients can confirm which node release they are talking to.

Result

NameTypeDescription
resultstringThe node's version string (e.g. "0.3.0-win.42"). The value is cacheable for roughly one day since the version does not change while the node runs.

Examples

DigiAssetWindows-cli.exe version

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "version", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
