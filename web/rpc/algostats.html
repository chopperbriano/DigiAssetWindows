algostats

algostats (start end timeFrame)

Returns per-algorithm mining difficulty statistics grouped into time frames over a given period. Warning: the last time frame in the result is likely not a full period.

Arguments

#NameTypeRequiredDescription
1startintegeroptionalStart time (Unix epoch seconds). Default is 0 (the beginning).
2endintegeroptionalEnd time (Unix epoch seconds). Default is the maximum value (the end).
3timeFrameintegeroptionalLength of each time frame in seconds. Default is 86400 (one day).

Result

A JSON array of objects, one per time frame. Each object has the following fields:

NameTypeDescription
timeintegerThe end time of the time frame (Unix epoch seconds).
algoarrayAn array with one entry per algorithm index. Each entry is either an object with the stats below, or null if that algorithm had no blocks in the time frame.
algo[].minfloatMinimum block difficulty for the algorithm in the time frame.
algo[].maxfloatMaximum block difficulty for the algorithm in the time frame.
algo[].avgfloatAverage block difficulty for the algorithm in the time frame.
algo[].countintegerNumber of blocks mined by the algorithm in the time frame.

If the requested data has been pruned, an empty array is returned.

Examples

DigiAssetWindows-cli.exe algostats 1620000000 1620086400 86400

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "algostats", "params": [1620000000, 1620086400, 86400]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
