getdgbequivalent

getdgbequivalent address index amount

Converts an amount denominated in an on-chain exchange-rate feed (identified by an oracle address and a rate index) into the equivalent value in DigiByte satoshis, using the latest rate the node has indexed. Throws if chain sync is more than 120 blocks behind.

Arguments

#NameTypeRequiredDescription
1addressstringrequiredThe oracle address that publishes the exchange rate.
2indexnumericrequiredThe rate index, an unsigned integer between 0 and 9.
3amountnumericrequiredFixed-precision amount to convert, with 8 decimals (unsigned integer).

Result

NameTypeDescription
resultnumericThe equivalent value in DGB satoshis (rounded up).

Examples

DigiAssetWindows-cli.exe getdgbequivalent dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh 1 100000000

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getdgbequivalent", "params": ["dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh", 1, 100000000]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
