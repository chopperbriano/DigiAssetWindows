listlastassets

listlastassets ( count firstAsset basic filter )

Lists the most recently issued DigiAssets in descending issuance order (newest first), starting from a given asset index. Optionally filters by Permanent Storage Pool (PSP) membership and returns either a basic summary or the full asset record.

Arguments

#NameTypeRequiredDescription
1countnumericoptionalMaximum number of records to return (must be at least 1). Defaults to unlimited.
2firstAssetnumericoptionalHighest asset index to start listing from (must be at least 1). Defaults to the newest asset.
3basicbooleanoptionalIf true (default) returns a basic summary per asset; if false returns the full DigiAsset JSON.
4filterobjectoptionalFilter object. Supports a psp member: a boolean (true = only assets in a PSP, false = only assets not in a PSP) or an integer (only assets in that specific PSP). Filtering may make the returned count differ from the requested count.

Result

An array. In basic mode each element is an object with the fields below; when basic is false each element is the full DigiAsset JSON (see getassetdata).

NameTypeDescription
assetIndexnumericNode-specific index of the asset (only valid on this node).
assetIdstringThe asset ID.
cidstringThe CID associated with the asset.
heightnumericThe block height at which the asset was issued.

Examples

DigiAssetWindows-cli.exe listlastassets 10

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "listlastassets", "params": [10]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
