asyncget

asyncget method (param1 param2 ...)

Retrieves the cached result of an async RPC job previously queued with asyncstart. Call it with the same method name and parameters that were passed to asyncstart; if the job has finished the stored JSON result is returned, otherwise false is returned.

Arguments

#NameTypeRequiredDescription
1methodstringrequiredName of the method that was queued with asyncstart.
2+param1, param2, ...anyoptionalThe same parameters that were passed to asyncstart, in the same order (used to identify the cached job).

Result

NameTypeDescription
(result)varies / boolThe result of the queued method, in the same format that method normally returns. If the queued method returned an object, an added cacheTime field (Unix epoch seconds) indicates when the result was cached. Returns false if the job has not completed (no cached result exists yet).

Examples

DigiAssetWindows-cli.exe asyncget algostats 0

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "asyncget", "params": ["algostats", 0]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
