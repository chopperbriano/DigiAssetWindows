getwalletbalances

getwalletbalances ( minconf )

Returns the wallet's spendable DigiByte balance together with every DigiAsset it holds, aggregated across all wallet UTXOs. Use getassetdata with an assetIndex to look up an asset's name and other metadata.

Arguments

#NameTypeRequiredDescription
1minconfnumericoptionalMinimum number of confirmations a UTXO needs before it is counted. Must be a non-negative integer. Defaults to 1.

Result

An object with the wallet's DigiByte balance and an array of the assets it holds. Note that the DigiByte total includes DigiByte locked in asset UTXOs, so the freely spendable amount will be less.

NameTypeDescription
digibyte.satsnumericDigiByte balance in sats (includes DigiByte locked in asset UTXOs).
digibyte.amountstringThe same DigiByte balance in decimal DGB.
assetsarrayOne entry per distinct asset held. Each entry has assetIndex (numeric), assetId (string), cid (string, the IPFS cid of the asset's metadata), decimals (numeric), count (numeric, amount held in smallest divisible units) and amount (string, amount held in display units).

Examples

DigiAssetWindows-cli.exe getwalletbalances 6

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getwalletbalances", "params": [6]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
