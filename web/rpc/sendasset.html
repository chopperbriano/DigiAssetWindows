sendasset

sendasset address asset amount ( options )

Sends DigiAssets from the wallet to an address. The wallet picks which asset UTXOs to spend, returns any asset change (of this or other assets carried on the same UTXOs) back to the wallet, and pays the transaction fee in DigiByte. Use sendtoaddress, not this method, to send DigiByte (assetIndex 1).

Arguments

#NameTypeRequiredDescription
1addressstringrequiredRecipient address or DigiByte domain.
2assetstring or numericrequiredAsset to send, given as an assetId (string) or assetIndex (integer). Assets with more than one index (unlocked assets whose id does not start with "Ua") must be identified by assetIndex.
3amountnumeric or stringrequiredAmount to send in display units (an integer, decimal number, or numeric string).
4optionsobjectoptionalObject with optional members: "changeAddress" (string) the address asset change should go to, defaulting to a new wallet change address; "dryrun" (bool, default false) build the transaction and return the cost breakdown WITHOUT broadcasting.

Result

By default the txid (string) of the broadcast transaction. When "dryrun" is true, an object describing the cost is returned instead of broadcasting.

NameTypeDescription
txidstringTransaction id of the broadcast send (default response).
outputsstringDry run only. Total value of the transaction outputs in decimal DGB.
estimatedMinerFeestringDry run only. Estimated miner fee in decimal DGB.
estimatedTotalstringDry run only. Estimated total cost (outputs plus miner fee) in decimal DGB.
satsobjectDry run only. The same "outputs" and "estimatedMinerFee" figures expressed in sats.
