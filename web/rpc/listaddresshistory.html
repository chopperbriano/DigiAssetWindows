listaddresshistory

listaddresshistory address (minHeight) (maxHeight) (limit)

Returns the list of transaction ids that a given address was involved in, optionally restricted by block-height range and count. Results are limited by the node's pruning level.

Arguments

#NameTypeRequiredDescription
1addressstringrequiredThe DigiByte address to look up.
2minHeightnumericoptionalLowest block height to include. Defaults to 1.
3maxHeightnumericoptionalHighest block height to include. Defaults to no upper limit. Must be greater than or equal to minHeight.
4limitnumericoptionalMaximum number of txids to return. Defaults to no limit.

Result

An array of transaction id strings that the address participated in.

NameTypeDescription
txidstringTransaction id the address was involved in (one entry per array element).

Examples

DigiAssetWindows-cli.exe listaddresshistory dgb1q...address

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "listaddresshistory", "params": ["dgb1q...address"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
