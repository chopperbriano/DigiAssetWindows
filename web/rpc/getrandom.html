getrandom

getrandom txid vout ( outOf ) | height ( outOf )

Returns a provably fair pseudo-random number for a transaction output or a block. The number is derived by SHA256-hashing the identifier together with the hashes of the 10 blocks up to and including its confirming block, then reducing the digest modulo an optional "out of" bound. Because every input is fixed on-chain data, callers can independently reproduce and verify the result. The confirming block must have at least 2 confirmations, and this only works if the underlying data has not been pruned.

Arguments

Two usages are supported. Usage 1 hashes a transaction output (first argument is a 64-character txid string); Usage 2 hashes a block (first argument is a block height integer).

#NameTypeRequiredDescription
1txid / heightstring or numericrequiredA 64-character txid (Usage 1) or a block height (Usage 2).
2vout / outOfnumericsee descriptionUsage 1: vout, the output index within the txid (required). Usage 2: outOf, the exclusive upper bound the result is reduced modulo (optional, defaults to the maximum 64-bit unsigned integer).
3outOfnumericoptionalUsage 1 only. The exclusive upper bound the result is reduced modulo. Defaults to the maximum 64-bit unsigned integer.

Result

The provably fair pseudo-random number.

NameTypeDescription
(result)numericAn unsigned 64-bit integer in the range 0 to outOf-1 (or 0 to 2^64-1 when no bound is given).
