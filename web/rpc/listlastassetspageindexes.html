listlastassetspageindexes

listlastassetspageindexes count ( filter )

Returns a paginated map of the most recently issued DigiAssets. Instead of the asset data itself, it emits compact {start, skips} index groups so a UI can page through assets by assetIndex.

Arguments

#NameTypeRequiredDescription
1countnumericrequiredNumber of records per page.
2filterobjectoptionalFilter object. Supports a psp member: a boolean (true = only assets in a PSP, false = only assets not in a PSP) or an integer (only assets in that specific PSP). Filtering may make the returned count differ from the requested count.

Result

An array of page objects, each of the form below.

NameTypeDescription
startnumericHighest assetIndex in this page group.
skipsarrayArray of assetIndex values that were skipped within this group.

Examples

DigiAssetWindows-cli.exe listlastassetspageindexes 25

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "listlastassetspageindexes", "params": [25]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
