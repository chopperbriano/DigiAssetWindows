getpsp

getpsp index

Returns the JSON description of a single Permanent Storage Pool (PSP) that this node knows about, selected by its index in the pool list. Throws if the index is out of range.

Arguments

#NameTypeRequiredDescription
1indexnumericrequiredThe PSP index number (unsigned integer) in the node's Permanent Storage Pool list.

Result

NameTypeDescription
resultobjectA JSON object describing the selected Permanent Storage Pool.

Examples

DigiAssetWindows-cli.exe getpsp 0

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getpsp", "params": [0]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
