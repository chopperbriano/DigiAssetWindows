getassetholders

getassetholders assetIndex|assetId

Returns an object listing every address currently holding the given DigiAsset together with the quantity each address holds.

Arguments

#NameTypeRequiredDescription
1assetIndex / assetIdnumeric or stringrequiredA numeric assetIndex or a string assetId identifying the asset.

Result

An object whose keys are holder addresses and whose values are the quantity held.

NameTypeDescription
<address>numericQuantity of the asset held by that address. One such key/value pair per holding address.

Examples

DigiAssetWindows-cli.exe getassetholders 1

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getassetholders", "params": [1]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
