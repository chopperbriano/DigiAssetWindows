getencryptedkey

getencryptedkey address

Returns the hex-encoded encrypted key blob(s) the node has stored for an address or list of addresses. Processing stops at the first address that has no stored value, so the returned array may be shorter than the input list.

Arguments

#NameTypeRequiredDescription
1addressstring or arrayrequiredA single address, or an array of address strings, to look up encrypted keys for.

Result

NameTypeDescription
resultarrayArray of hex-encoded encrypted key strings, one per address, in input order, up to the first address with no stored value.

Examples

DigiAssetWindows-cli.exe getencryptedkey dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id": "curltest", "method": "getencryptedkey", "params": ["dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
