getwalletbalances

Returns the wallet's spendable DigiByte balance and all DigiAssets it holds. Use getassetdata with an assetIndex to look up an asset's name and other metadata. Requires DigiByte Core to be built with wallet support.

Input Parameters

minconf (integer, optional, default 1): Minimum confirmations a UTXO needs to be counted.

Output Format

{
    "digibyte": {
        "sats": 12345678900,        // spendable DigiByte in sats(includes DigiByte
                                    //     locked in asset UTXOs)
        "amount": "123.45678900"    // same value in decimal DGB
    },
    "assets": [
        {
            "assetIndex": 2,
            "assetId": "La....",
            "cid": "bafk....",      // IPFS cid of the asset's metadata
            "decimals": 2,
            "count": 150,           // amount held in smallest divisible units
            "amount": "1.50"        // amount held in display units
        },
        ...
    ]
}

Examples

digiasset_core-cli getwalletbalances

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getwalletbalances", "params": []}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
