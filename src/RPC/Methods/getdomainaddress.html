getdomainaddress

Returns the DigiByte address associated with a specific domain. This command is useful for resolving the blockchain address linked to a domain name registered in the DigiByte network.

Input Parameters

domain (string): The domain name for which the associated DigiByte address is requested.

Output Format

String representing the DigiByte address associated with the given domain.

Errors

Throws an RPC error (code -1, RPC_MISC_ERROR) with one of the following messages when the domain cannot be resolved:

  • Unknown Domain — the domain has never been issued (it is available to register).
  • Domain Revoked — the domain existed but was revoked and no longer resolves.
  • Domain Burned — the domain is registered but its controlling asset has no holders (it was swept by a non-DigiAsset-aware wallet), so it can no longer resolve.

Examples

digiasset_core-cli getdomainaddress "example.dgb"

curl --user myusername --data-binary '{"jsonrpc": "1.0", "id":"curltest", "method": "getdomainaddress", "params": ["example.dgb"]}' -H 'content-type: text/plain;' http://127.0.0.1:14024/
