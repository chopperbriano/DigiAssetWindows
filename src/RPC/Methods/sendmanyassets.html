sendmanyassets

sendmanyassets

Sends DigiAssets to several recipients in a single transaction. The wallet automatically picks which asset UTXOs to spend, sends all asset change back to the wallet and pays the transaction fee in DigiByte. Requires DigiByte Core to be built with wallet support and the wallet to be unlocked.

Input Parameters

sends (array): one entry per recipient, up to 31 entries (protocol limit - asset outputs must sit in the first 32 outputs). 

[{
    "address": "dgb1....",        // recipient address or DigiByte domain
    "asset": "La....",            // assetId or assetIndex.  unlocked assets
                                  // not starting with "Ua" must use index
    "amount": 5                   // amount in display units
}, ...]

options (object, optional): 

{
    "changeAddress": "dgb1...."   // address asset change should go to.
                                  // defaults to a new wallet change address
}

Output Format

txid (string) of the broadcast transaction

Examples

digiasset_core-cli sendmanyassets '[{"address":"dgb1qtqt4vrsjfnncr7wjvdvhw7evgzsyj39kaxhg6z","asset":"La5x1DEP8J2XiViw9J7bhCFXVsJnfLDBBWo6aS","amount":5},{"address":"dgb1qunxh378eltj2jrwza5sj9grvu5xud43vqvudwh","asset":5350,"amount":2}]'
